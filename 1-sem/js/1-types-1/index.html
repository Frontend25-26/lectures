А что насчет var?

							
								Устаревший способ объявления переменных
								выполняется всплытие переменных (подробнее в следующем уроке)

Операторы присваивания и операторы сравнения

тип: symbol

							
								Символ (анг. Symbol) — это уникальный и неизменяемый тип данных,
								который может быть использован как идентификатор для свойств объектов

тип: symbol

							
								//  Symbol уникален, даже если создан с одинаковым описанием:

								const sym1 = Symbol('описание');
								const sym2 = Symbol('описание');

								sym1 === sym2; // false - всегда уникальны!

тип: symbol

								const id = Symbol('id');

								const user = {
									[id]: 123,
									name: 'John'
								};

тип: symbol

							
								// Избежание конфликтов имен

								const user = {};

								// Service A
								{
									const id = Symbol('id');
									user[id] = 'Service A';
									console.log(user[id]); // Service A
								}

								// Service B
								{
									const id = Symbol('id');
									user[id] = 'Service B';
									console.log(user[id]); // Service B
								}

								console.log(user); // { [Symbol(id)]: 'Service A', [Symbol(id)]: 'Service B' }

итого

							
								примитивы
								number - для чисел, ограничены диапазоном ±(2**53-1)
								bigint - для целых чисел произвольной длины
								string - для строк, содержит ноль или больше символов
								boolean - для определения true или false
								null - для неизвестных значений
								undefined - для неприсвоенных значений
								symbol - для уникальных идентификаторов

Преобразование к Boolean

							
								Операторы вроде || и && выполняют преобразование значений
								к Boolean, а возвращают значения исходных операндов

								let a = '' && 123;        // a === '' (первое falsy)
								let b = 2 && 123;         // b === 123 (последнее truthy)

								let c = 'hello' || 123;   // c === 'hello' (первое truthy)
								let d = false || 123;     // d === 123 (первое truthy)

итого

							
								Примитивные типы:
								number - для чисел, ограничены диапазоном ±(2^53 - 1)
								bigint - для целых чисел произвольной длины
								string - для строк, содержит ноль или больше символов
								boolean - для определения true или false
								null - для неизвестных значений
								undefined - для неприсвоенных значений
								symbol - для уникальных идентификаторов

								Не примитивный тип:
								object

								Структуры данных:
								function
								array
